# Project OSSEC: Network Traffic Anomaly Defender

This repository contains a **network traffic anomaly detection system** combining:
- a **Python backend** for NetFlow parsing, feature extraction, and ML-based detection
- an **Angular frontend** for visualization and interaction

---

## Interface (Angular Frontend)

This interface was generated using  
[Angular CLI](https://github.com/angular/angular-cli) version **21.0.4**.

### Development server

To start a local development server, run:

ng serve
